======================================================================================================

Controle Econômico v.1.0.0.

======================================================================================================

O programa Controle-Economico foi desenvolvido exlcusivamente para sistemas operacionais Windows x64 
bits, e opera via terminal CMD, seu uso de destina a aplicações que não exijam alto desempenho ou 
segurança. Para preenchimento de campos de texto basta digitar de forma livre e pressionar enter 
quando quiser que o texto seja encerrado, em opções de seleção de múltiplas opções, as teclas de 
orientação são: seta para cima e seta para baixo, também é necessário pressionar a tecla enter quando
chegar ao item desejado.
Para que a exibição do conteúdo dos campos de texto seja totalmente legíveis não é recomendado o uso 
de acentos ou caracteres especiais(-´~1?/+_). 
Os menus de seleção indicam as opções de controle via teclado(números,ESC,ENTER). 
Na ficha de preenchimento os campos: descrição e detalhes possuem respectivamente um limite de 100 e
500 caracteres, e quando o limite de caracteres é atingido a caixa de textos encerra o preencimento, 
e passa para o item seguinte na ficha. 
Em caso de alteração de datas deve-se atentar ao preenchimento da mesma, por se tratar de uma string, 
qualquer valor pode ser adicio-nado porém não é recomendado o uso de letras, sinais ou caracteres 
especiais (a,s,g,*,@) assim como teclas especiais do teclado(enter,esc,backspace). 
O arquivo main_file.exe é o executável, caso não funcione será necessária uma nova compilação no 
ambiente que ele irá trabalhar.
Este é um software livre criado para facilitar a organização econômica pessoal e com fins didáticos.

======================================================================================================
buble_sort:
Bibliteca que contém ferramentas para ordenação de listas dinâmicas.

toolslibrary:
Biblioteca que contém ferramentas de busca e edição de cadastros.

loadlibrary:
Biblioteca que contém ferramentas de interação no sentido arquivo->lista dinâmica e vice versa,
possui ferramentas para desalocação de memória, e alternância entre arquivos da base de dados.

cadlibrary:
Função que possui ferramentas para receber dados em uma estrutura e organizá-los em uma lista.

Database:
Diretório de armazenamento da base de dados, que possui subdiretórios para armazenamento dos dados 
de cada ano, estes subdiretórios por sua vez, armazenam os arquivos de cadastro de cada mês.

main_file:
Código fonte do programa, possui a função principal(main()) entre outras funções e bibliotecas
